Command Line Interface
======================

Docker Usage
~~~~~~~~~~~~

To list available commands, either run ``docker`` with no parameters or execute
``docker help``::

  $ docker
    Usage: docker COMMAND [arg...]

    A self-sufficient runtime for linux containers.

    ...
